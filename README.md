# Minimalistic config for nvim by segfaultchan
## dependencies
- `nvim` - editor
- `unzip` - for mason
- `npm` - also for masom
- `tar` - also for mason
- `rust` or `rustup` - for rust lsp
- `go` - for go lsp
- `curl`
- `ueberzugpp` - (optional) for image.nvim (check lua/plugins/helpful_plugins.lua - backend)
- `imagemagick` - (optional) also for image.nvim
